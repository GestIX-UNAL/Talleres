# Subsistema 3: Visualización 3D optimizada (Three.js + AR.js).

* Escena principal en Three.js / React Three Fiber con overlays dinámicos.
* Implementación de modelos 3D interactivos o animados.
* Integración AR.js con marcadores personalizados.

## Marcadores personallizados

Usamos https://jeromeetienne.github.io/AR.js/three.js/examples/marker-training/examples/generator.html para personalizar el marcador AR `.patt`. 

## Modelos animados

Utilizamos un modelo GBL de una araña en https://poly.pizza/m/yRYJiAJyiM para obtener el modelo `Spider.glb` con animaciones.
